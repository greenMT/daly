
# Daly VM


## Function

* name
* number of arguments
* number of locals
* labels
* instructions


## Instructions

* `CALL <fn>`

* `RETURN`

* `CMP [EQ, LT, LE, GT, GE]`

* `JMP <label>`

* `JMP_IF <label>`

* `STORE <target>`

* `LOAD <target>`

* `CONST <number>`

* `ARRAY <size>`

* `LEN`

* `PRINT`

* `CLONE`

* `LOOP`
    marks a loop header

* `BREAK`
    marks end of a loop
